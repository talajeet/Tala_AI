# Tala_AI
يهدف هذا المشروع إلى تطوير وتقييم نموذج ذكاء اصطناعي للتعرف على الأرقام المكتوبة بخط اليد باستخدام مجموعة بيانات Digit Recognizer. تم بناء نموذج أساسي باستخدام Logistic Regression ثم تحسين الأداء باستخدام شبكة عصبية متعددة الطبقات (MLP)، مع تحليل النتائج ومقارنتها للتحقق من دقة النموذج وقدرته على التعميم.
